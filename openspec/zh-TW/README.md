# OpenSpec 繁體中文文件目錄

本目錄為 OpenSpec 文件的固定繁體中文位置。

## 固定路徑

- 根目錄：`openspec/zh-TW/`
- 規格：`openspec/zh-TW/specs/`
- 變更文件：`openspec/zh-TW/changes/`

## 使用規則

1. 新增或修改 OpenSpec 文件時，需同步更新本目錄對應檔案。
2. 若原始 OpenSpec 檔案路徑為 `openspec/...`，繁中版本需鏡像到 `openspec/zh-TW/...`。
3. 文件同步需符合 `documentation-governance` 與 Doc Sync Gate 要求。

## 維護建議

- 先更新原始變更內容，再同步更新繁中版本。
- 每次變更後，以關鍵字檢查是否殘留英文模板語句。
- 在 PR 審查時，將本目錄納入必檢項目。
