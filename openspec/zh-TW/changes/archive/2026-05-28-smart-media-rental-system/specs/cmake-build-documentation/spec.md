## 新增需求

### 需求：CMake 外部建置
系統必須可使用 CMake 標準 out-of-source 流程成功編譯。

#### 情境：在 build 目錄內編譯專案
- **當** 開發者在獨立 build 目錄執行 `cmake ..` 與 `cmake --build .`
- **則** 專案必須可無警告編譯完成，並輸出可執行目標 `SmartLibrarySystem`。

### 需求：測試資源自動複製
建置系統必須在 CMake 設定或建置階段，自動將初始 mock 資料目錄（`data/`）複製到輸出目錄。

#### 情境：直接從 build 目錄啟動執行檔
- **當** 使用者直接在 build 目錄中執行編譯後二進位檔
- **則** 系統必須可從本地 `data/` 目錄成功載入設定資料，且不發生路徑錯誤。
